RÈGLES D'ARCHITECTURE (critiques) :
1. TOUT passe par core/hub.py — aucun module ne parle à un autre directement
2. La mémoire n'est écrite QUE par le worker (memory_worker()) — jamais par le LLM de chat
3. Les outils sont déclarés dans NIMM_TOOLS et exécutés via _execute_tool()
4. Les embeddings sont calculés avec _embed() — jamais directement dans un module
5. Le system prompt est assemblé par build_system_prompt() — ordre strict
6. Les imports : d'abord les librairies tierces, puis les modules internes
